== Description ==
This is a Chiropractor theme with with many options for customization without having to know any code. It's great for chiropractic, physical therapy, massage therapy, dental, acupuncture or naturopathic.

== Frequently Asked Questions ==

= How to I use this theme =

1. 	Upload the chiro-pro.zip file from within the Themes area of your WordPress dashboard.

2. 	Select Chiro Pro and click activate.

3.	Go to http://chirothemes.com and watch the videos related to your theme.
